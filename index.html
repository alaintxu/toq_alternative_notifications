Welcome Toq Alternative Notifications App's Page

Toq Alternative Notifications (a.k.a. ToqAN) is an app that enables Android Lollipop notifications to be sent properly to Qualcomm's Toq Smartwatch. Qualcomm's Default Toq App is not being maintained anymore, and it does not work properly with Android Lollipop. All the text of the notifications are sent in the title section. This app sends those notifications properly and enables an applet with the devices current notifications.

How to use it

This app depends on Qualcomm's Default Toq App.

  1. Install Qualcomm's Default Toq App.
  2. Pair your device with the Smartwatch.
  3. Go to "Manage Notifications" and disable all notifications not to have double notifications.
  4. Install Toq Alternative Notifications (ToqAN).
  5. On ToqAN, click on "Install Deck Of Cards" and enable the Applet.
  6. Click on "Check Notification Permissions" and make sure that ToqAN checkbox is enabled.

Now, all notifications should be sent to the smartwatch.

FAQ

1. The notifications do not appear in the Smartwatch, what can I do? Make sure that you made all the steps on "How to use it". If you continue without receiving notifications, click on "Check Notification Permissions", Disable the ToqAN checkbox and enable it one more time. It should work now.
